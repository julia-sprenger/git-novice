## Installing Git

This lesson requires git. Check the workshop setup instruction on how to install `git bash`.

## Algobank Account & Carpentries Group

You will need access to your [OECD Algobank](https://gitlab.algobank.oecd.org/) account to follow episodes 7 & 8 in this lesson.

1. Go to <https://gitlab.algobank.oecd.org/>
2. Sign in with your OECD credentials. Note that you might need to be connected to the OECD network, use the OECD VPN if you can't sign in from another network.
3. In the navigation bar select `Groups`. The list of groups accessible for you should contain the group dedicated for this lesson: [CARPENTRIES_PYTHON](https://gitlab.algobank.oecd.org/CARPENTRIES_PYTHON)

![](fig/Algobank_group.png)

::::::::::::::::::::: callout

To be able to log sign in the Algobank you will need to be member of at least one OECD Data project. For more details on Data projects follow the [Working with Data at the OECD E-learning course](https://oecdcampus.oecd.org/Training/view/4473516).
